# 360浏览器批量下载图片

# 添加图片下载插件

  • 打开360浏览器，点击右上角插件管理按钮 -》添加
  • 进入360应用市场，搜索"图片批量下载神器"，点击进入安装
  • 安装完成，右上角就有了图片下载图标

# 图片下载插件设置

  • 设置打包下载
  • 设置WEBP转JPG
    • 点击设置
    • 开启

# 图片下载

  • 京东搜索到商品后，右键“审查元素”，模拟手机模式打开
  • 鼠标滚动到最下面，待商品详情图片加载完成后，点击图片下载插件，【全选/下载】
  • 下载后的压缩包里面的内容
